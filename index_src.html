Security, Data Sovereignty & AI Disclaimer

AI-Free Processing Statement: This application is completely free of any internal or external artificial intelligence (AI), machine learning (ML), or Large Language Model (LLM) processing. All telemetry analysis, remediation recommendations, and printed action plan reports are compiled locally using deterministic, static JavaScript logic. No data is sent to third-party AI or LLM systems.

Data Sovereignty & Privacy: The tool runs entirely in your local browser sandbox and is 100% self-contained. Your API refresh tokens, settings configurations, and custom logistics databases are stored locally in your browser's private localStorage partition. No analytics scripts, cookies, or external trackers are loaded.

Policy Compliance: Access is strictly read-only. All metrics are retrieved directly from NetApp's official, TLS-secured Active IQ API endpoint (activeiq.netapp.com). ARIA is an independent, third-party tool and is not affiliated with or endorsed by NetApp, Inc.
