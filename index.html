API Setup Guide

1 Log in to Portal
Open activeiq.netapp.com and sign in with your NetApp Support Site (NSS) credentials.
2 Generate API Token
Navigate to Quick Links → API Services → Generate Token.
3 Paste & Save
Paste the Refresh Token into the API Authentication card above, then click Save Configuration.
4 Sync Data
Click Sync Now. Watchlists are auto-discovered from the API — no manual ID entry needed.
